・ファイル名について
/////全て小文字でアンダースコアで繋げる
例：charadata.cpp game_screen.cpp

/////ファイルのcppとhは各属性に分ける

・クラス名について
/////頭文字を大文字、それ以降の単語区切りも大文字で記載
例：class EnemyData : public CharaData

・メソッド、関数名について
/////頭文字を小文字、それ以降の単語区切りは大文字で記載
例：void checkHit();
    void setX(float pos_x){x = pos_x};

・変数名について
/////全て小文字、単語区切りをアンダースコアでつなぐ
名前の最初に各属性の頭文字を小文字、その次にアンダースコアをつける
例：float x;
    float y;

・警告は無くす


関数の引数にクラスを渡すとき
4バイトを超えるデータの時は参照・ポインタ

例：
class MyClass 512byte
{
	MyClass my:
}

void func(Myclass &a){
	
}

constをつけると書き換わらない
void func(const Myclass &a){
	
}
